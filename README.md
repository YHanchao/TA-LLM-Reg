# TA-LLM-Reg：LLM辅助R语言数据分析

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 📖 项目简介

本仓库是**2025年春季学期《回归分析与数据思维》课程**的助教课代码仓库。课程面向北京大学光华管理学院学生。

**主要内容**：探索如何使用大语言模型（LLM）辅助R语言进行数据分析，涵盖回归分析的核心概念与实战应用。

## 📊 数据集说明

本仓库代码使用的数据集来源于**《（新版）商务数据分析与应用》**教材配套资源。

- **数据来源**：狗熊会慕课平台
- **数据链接**：[https://www.xiong99.com.cn/p/t_pc/course_pc_detail/image_text/i_5edafde30594b_kkXYFFvK](https://www.xiong99.com.cn/p/t_pc/course_pc_detail/image_text/i_5edafde30594b_kkXYFFvK)
- **版权声明**：数据集版权归原作者/狗熊会所有。本仓库**仅包含代码**，不包含任何原始数据。用户需自行通过上述链接获取数据。

> ⚠️ **重要提示**：仓库中的 `.csv` 文件均为**空白占位文件**，仅用于展示数据结构。运行代码前，请务必从上述链接下载完整数据集，并替换这些空白文件。

## 🚀 快速开始
1. **克隆本仓库**
   ```bash
   git clone https://github.com/YHanchao/TA-LLM-Reg.git
   ```

2. **下载数据集**
   - 访问[课程链接](https://www.xiong99.com.cn/p/t_pc/course_pc_detail/image_text/i_5edafde30594b_kkXYFFvK)
   - 获取数据文件（提取码：`fj0r`）
   - 将下载的数据文件**替换**对应课时文件夹中的空白 `.csv` 文件

3. **运行代码**
   - 打开任意课时的 `.Rmd` 文件
   - 在RStudio中点击 `Knit` 即可生成对应的 `.html` 报告

## 💡 使用建议

### 对选课同学
- 动手实践：不要只查看 `.html` 报告，务必运行 `.Rmd` 源文件，尝试修改参数
- 数据准备：每次运行前，确保对应文件夹中的 `.csv` 文件已替换为真实数据
- 遇到问题：欢迎在 Issues 中讨论，或联系助教团队

### 对助教团队成员
- 代码更新：直接推送前请先 `git pull` 拉取最新代码
- 提交规范：commit信息请简明扼要（如“lesson3: 添加交互项示例”）
- 分支管理：如有较大改动，建议先创建分支，合并前通过 Pull Request 讨论

欢迎各位助教同学完善代码、添加示例、修正错误！
